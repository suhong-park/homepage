이런 문제에 막힙니다

📂

문서는 많지만 찾기 어렵다

사내 자료가 부서별로 흩어져 있어 필요한 정보를 찾는 데 과도한 시간이 소요됩니다.

🤖

ChatGPT는 쓰지만 회사 기준으로 답하지 못한다

외부 AI는 내부 문서와 업무 기준을 모릅니다. 사내 지식 기반의 정확한 답변이 필요합니다.

🔒

보안 때문에 내부 문서를 올리기 어렵다

민감 데이터를 외부 AI 서비스에 입력하는 것에 보안·컴플라이언스 우려가 있습니다.

🔄

어떤 AI부터, 어떻게 시작해야 할지 모른다

반복 업무는 많지만 어떤 자동화를 어떤 순서로 도입해야 효과가 나는지 판단이 어렵습니다.

→ 워크스페이스 + Named User

FAQ

자주 묻는 질문

내부 문서가 외부로 유출될 가능성은 없습니까?
centricAI Suite는 온프레미스 또는 Private Cloud 환경으로 구축할 수 있습니다. 이 경우 문서, 질의, 응답 데이터는 고객사 내부 통제 범위 안에서 운영됩니다. 사용자 권한 관리와 Audit Log를 통해 모든 접근 이력을 추적·관리할 수 있습니다.
AI 답변을 그대로 업무에 사용해도 됩니까?
업무 담당자의 검토를 전제로 활용하는 것이 적절합니다. centricAI Suite는 정보 탐색과 초안 작성 시간을 줄이는 역할을 하며, 최종 판단은 담당자가 수행하는 구조를 권장합니다. 출처·페이지 인용을 함께 제공해 검토 효율을 높입니다.
기존 ERP, 그룹웨어 시스템과 연동됩니까?
연동 가능성은 고객사의 시스템 구조, API 제공 여부, 보안 정책에 따라 달라집니다. centricAI Suite는 Tool Connector 및 MCP 기반 확장 구조를 통해 연동을 검토할 수 있으며, 실제 적용 범위는 PoC 단계에서 함께 확인합니다.
비용 절감 효과를 보장할 수 있습니까?
도입 즉시 비용 절감을 보장하기보다, 반복적인 문서 검색·요약·데이터 조회 업무에서 시간 절감 가능성을 PoC로 먼저 검증합니다. 이후 검증된 업무부터 단계적으로 확장해 ROI를 명확하게 산정할 수 있습니다.
영구 라이선스와 구독 라이선스의 차이는 무엇입니까?
영구 라이선스는 1회 구매 후 계속 사용할 수 있으며, 구독 라이선스는 연간 계약 방식으로 초기 비용 부담이 낮습니다. 포털 Standard 기준 영구 3,000만원, 구독 780만원/년이며, Named User는 영구 300만원, 구독 78만원/년입니다. 도입 규모와 예산에 맞는 방식을 PoC 상담 시 함께 검토합니다.
